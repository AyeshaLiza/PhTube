  1. 1. Discuss the scope of var, let, and const

    2. Ans: Var is Global or functional Scoped. let & const both can be block or global scoped. Const takes a constant variable which which can neither be redeclared nor be updated. Whereas the variable declared with let can be update but not redeclarable

    3. Ans:These 2 are primitive data types. in some cases these are found:
    Null: Null is a pre-defined or assigned value in a variable when there is absence of any value. undefined: it refers that, a variable which is declared but not assigned yet, that is called undefined.
